# 信息安全基础知识



五个基本要素

1. 机密性：确保信息不暴露给未授权的实体或进程。
2. 完整性：只有得到允许的人才能修改数据，并且能够判断出数据是否已被篡改。
3. 可用性：得到授权的实体在需要时可访问数据，即攻击者不能占用所有的资源而阻碍授权者的工作。
4. 可控性：可以控制授权范围内的信息流向及行为方式。
5. 可审查性：对出现的信息安全问题提供调查的一句和手段。





# 加密技术与认证技术



 对称加密（大量）：DES、AES、3DES、RC-5、IDEA算法。口诀：3S+51

​	分组加密：DES、AES、3DES、IDEA

​	流密码：RC-5



非对称加密：RSA、DSA、ECC。口诀：AA、CC





数字签名主要用于认证（不可否认性），身份认证使用数字证书。

信息摘要主要用户验证完整性。
