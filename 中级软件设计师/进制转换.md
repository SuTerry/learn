# 加权法/按权展开法（其他进制转十进制）



将R进制的每一位数值用R<sup>k</sup>的形式表示。R为机制的数字，k为位数。



111B转10进制

1X2<sup>2</sup>  1X2<sup>1</sup> 1X2<sup>0</sup>

4	2       1

1        1       1

4+2+1 = 7D



3B.04H转10进制

3X16<sup>1</sup> 11X16<sup>0</sup> 0X16<sup>-1</sup> 4X16<sup>-2</sup>

48       11.       0	  0.015625

3	 B.	  0	  4



48+11.+0+0.015625 = 59.015625D





# 短除法/除基取余法（十进制转其他进制）



循环除基直到商为0，从下往上取余。



94D转2进制

2|94	余 0

2|47	     1

2|23	     1

2|11	     1

2|5  	     1

2|2  	     0

2|1  	     1

​    0

得到结果：1011110B



139D转8进制

8|139	余3

8|17	  1

8|2	    2

​    0

得到结果：213O

# 码制转换



原码：最高位是符号位，其余低位表示数值的绝对值，<span style="color: red">直接表示该数据</span>（0表示整数，1表示负数）

反码：<span style="color: red">正整数的反码与原码相同</span>，负数的反码是在负数原码基础上数值位<span style="color: red">取反</span>（符号位不变）

补码：<span style="color: red">正整数的反码与原码相同</span>，负数的补码是在负数反码基础上末位<span style="color: red">补1</span>（符号位参与运算）

移码：无论正负数，都是把补码的<span style="color: red">符号位取反</span>
